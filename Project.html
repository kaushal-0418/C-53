React Native

What is React Native?

React Native is an open-source mobile application framework created by Facebook.It is used to develop applications Android,IOS,Web and UWP. Skype,Instagram,Tesla,Walmart are some apps made using React Native. React Native uses a mix of JavaScript and html languages to code, this combination is called JSX.

About React Native

React Native is based on React.js, which is written in JavaScript.It is simple to learn. You have functions like 'import' and 'export'.Import function brings into app.js which is where the main code is located. You can create other files and simply import them to app.js.To add sound you have 'playSound' and for images it is 'image'. You can display evryhting using render function, render displayes evryhting that you return. It uses multiple html tage and codes to design and create apps. Most commonly used tags are Text, View and compnents like style are also commonly used.
